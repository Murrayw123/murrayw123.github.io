Projects

Portfolio Site

My personal site I put together to showcase some of the things I'm currently doing and am currently interested in. When building the site, I primarily used the Materialize Responsive CSS framework. I have nothing but nice things to say about it. The grid is really lovely to work with and the helpers they include such as footer formatting, form items and buttons make life a lot easier.

The Big Lap

A webapp I built using Python and Flask when I was planning a trip across the Nullabor last year. I thought it would be a good idea to put together a site that calculates fuel usage, fuel cost and driving time. First time hosting anything on Heroku and I'm impressed! In the future I will be fixing up the responsive nature of the page and implementing a login module to allow the saving of trips.

Crypto Search

A webapp I built using React and Ant Design. Pulls in the top cryptocoins 8 as a home page and allows you to search through the top 100 to get some price information and a pretty graph. With props to CoinMarketCap for the API.
