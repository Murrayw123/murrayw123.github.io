Projects

The Big Lap

A webapp I built using Python and Flask when I was planning a trip across the Nullabor last year. I thought it would be a good idea to put together a site that calculates fuel usage, fuel cost and driving time. First time hosting anything on Heroku and I'm impressed! In the future I will be fixing up the responsive nature of the page and implementing a login module to allow the saving of trips.

Crypto Search

A webapp I built using React and Ant Design. Pulls in the top cryptocoins as a home page and allows you to search through the top 100 to get some price information and a pretty graph. With props to CoinMarketCap for the API.

Munda Biddi Map

A webapp built for the Munda Biddi Trail Foundation. Built using Django, SQLite, Javascript and the Google Maps API this application calculates distances between major points of interest that foundation members can add through Django Admin.

Bulk Order Recipes

A web app created for schools and organisations that need to scale recipes from standard recipes one might find in a recipe book, converts all imperial measurements and then combines them for a final food order. Currently scales by a total number and a group (eg order for 30 or order for 2 groups of 5 students) but will eventually add the ability to add a class.

Front end built using Javascript ES6 and React. Backend is a Postgres/Golang API. The app features a full (and mostly working!) authentication system that hashes and stores passwords and sends out verification and reset emails.
